## 📖 About

This is a program for an GUI based digital clock

---

## 🚀 Getting Started

### Prerequisites

- Python 3.6 or higher

- PyQt5 library version 5.15.11 or higher

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/TheDevGod-Build/digital_clock.git
   cd alarm_clock
   ```
2. **Install PyQt5**
   ```bash
   pip install PyQt5
   ```
3. **Run the program**
   ```bash
   python main.py
   ```
